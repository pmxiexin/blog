《 奈飞文化手册》读书笔记

发表于 2019-02-10 | 分类于 读书笔记
字数统计: 3.6k | 阅读时长 ≈ 12 分钟

读后感悟

得知这本书是在一个公众号上面，他引用了奈飞文化准则的第一条“只雇佣成年人”。看到这句话就感觉心中有什么被击中了，天哪，我头疼了半天的原来是这个。
为什么我要和那么多“不成年人”相处啊。为什么要监督别人认真工作，按时上班。为什么要容忍他人频频互相吵架，频频擅改需求，频频隐瞒问题。出了问题只能安慰自己说“你应当变得包容点，学会和各种各样的人相处，这是一种能力哦”。但让我总是将精力投入在这些调和关系的事上，希望又不影响项目又不让对方难堪。
因为太多人都“不成年啊”。口无遮拦、意气用事、擅作主张，这不应当出现在一个成年人身上。成年人懂得责任，为自己的行为负责；成年人懂得自省，会进行自我进化。
于是我马上开始阅读了这本书。

这本书是一本好书，客观地分析了各个文化准则产生的背景及影响。就是受众面太过狭窄，不具备普适性。真的只是“奈飞文化手册”啊，或是叫做“顶级企业和顶级员工可遵循的文化手册”。

大多数人力资源管理或是企业文化建设，都是基于“普通人”的，而这本书的是基于“成年人”的，。两者的差异如下：

  1. “普通人”：职业技能一般；不会自我进化，需要他人帮助；充满人性的弱点…
  2. “成年人”：职业技能出色；会独立的自我进化；能克服人性的弱点……
    正是因为这一类的差异，让传统管理方式和奈飞管理方式产生极大差异：
  3. 传统的管理方式是“雇佣”：公司是主人，安排员工的工作。一层层的职级差异，一道道的规则限制，都是为了监督员工行为，避免人为风险。
  4. 奈飞的管理方式是“依靠”：员工是主人，决定公司走向。减少职级差异，减少规则限制，都是为了发挥员工能动性，多用脑子。

典型的差异是在回答这三个问题的时候

  1. 为什么我在这家公司：
    1. 传统：因为这家公司，无论是薪酬还是福利，都让我满意
    2. 奈飞：因为这家公司和里面的同事，做着我喜欢的事情
  2. 为什么我要工作：
    1. 传统：因为公司制定的KPI，我要完成它，不然会丢饭碗
    2. 奈飞：因为我喜欢，工作让我兴奋
  3. 为什么要给薪水：
    1. 传统：因为公司雇用我干活，占用了我的时间和劳力，就得给薪水
    2. 奈飞：因为我的工作恰好能让公司获得收益，公司应给予报酬
      正是这些差异，让奈飞的文化无法适用于大多数企业。毕竟多数人都是充满缺点，人性复杂的普通人。

除了适用群体的差异外，这本书在国内的应用，还有着东西方文化的差异。
比如在看待“企业发展后，是培养旧人还是招募新人”的问题上，奈飞认为应该招募新人，因为他信任的是市场，市场上会有更合适的人，旧人不适合新的工作就应该离开。而国内很多公司会认为应该培养旧人，比如阿里巴巴的创始人们，都身居要位，可他们并不一定是当时最有能力的。因为国内信任的是人脉关系，相信我们之间的信任会激发出感恩的Buff——你器重我，我就加倍回报你。

总体来说，这是一本有想法，有深度的管理书籍。但是不能盲目的应用，应当有所甄别的，谨慎使用。最重要的是，两句话我永远记着

  1. 我们只雇佣、奖励和容忍完全成熟的成年人
  2. 人们最希望从工作中得到的东西，是加入到让他们信任和钦佩的同事团队中，大家一起专注于完成一项伟大的任务。

书本内容

第一章：只雇用、奖励和容忍完全成熟的成年人

内容概要

  1. 只雇用、奖励和容忍完全成熟的成年人。
  2. 让员工加入到让他信任和钦佩的同事团队中。
  3. 打造尽可能简洁的工作流程和强大的纪律文化。
  4. 不要让规章和制度限制了高绩效者

观点摘录：

  1. 人们最希望从工作中得到的东西，是加入到让他们信任和钦佩的同事团队中，大家一起专注于完成一项伟大的任务，所以如果企业能够为员工提供这样的机会，那么他们就会更愿意加入到企业中来。
  2. 伟大的团队是这样的团队：其中的每一位成员都知道自己要前往何方，并愿意为此付出任何努力。建立伟大的团队不需要靠激励、程序和福利待遇，靠的是招聘成年人，渴望接受挑战的成年人，然后，清晰而持续地与他们沟通他们面对的挑战是什么。
  3. 人都是拥有权力的。企业的任务不是要对员工赋能，而是要从员工踏进公司大门的第一天起，就提醒他们拥有权力，而且为他们创造各种条件来行使权力。
  4. 管理者的本职工作是建立伟大的团队，按时完成那些让人觉得不可思议的工作。只有这一项工作是管理应该做的。

第二章：如果能够很好地理解公司的业务，高绩效者就能够更好地工作

内容概要

  1. 建立新员工大学，保持沟通的强节奏。
  2. 双向沟通，为员工提供向所有管理者提问的机会。
  3. 让每一位员工了解，他为客户带来的体验是如何直接影响公司利润的。
  4. 如果只选择一门课程面向公司全员开授，请选择公司业务运作和客户服务的基本知识。
  5. 最好的福利，是让员工有机会去更好地了解业务和客户。

观点摘录：

  1. 我们要求开放、清晰和持续地沟通工作任务以及面临的挑战，这个要求不是针对某个管理者的团队的，而是针对整个公司的。
  2. 最好不要去臆想员工很笨，而是要考虑到另外一种情况：如果员工做了愚蠢的事情，要么是未被告知相关信息，要么是被告知了错误信息。
  3. 员工应该了解，永远不要向上级主管隐瞒问题或信息。作为领导者，你应该身先士卒，用行动而不是用语言来表明你希望员工可以畅所欲言，他们可以直接告诉你业务相关信息，哪怕是坏消息或是你不赞同的消息。否则，大部分人永远都无法真正向你敞开心扉。
  4. 如果你想知道员工在想什么，没有比直接询问他们更好的方法了，而且最好是当面询问。

第三章：绝对坦诚帮助人们成长

内容概要

  1. 人前人后要做到言行一致。
  2. 公开批评，面对面沟通是解决问题的最有效方法。
  3. 只有当管理者能够坦承错误时，员工才能够畅所欲言。
  4. 公司要有一套透明的反馈系统。

观点摘录

  1. 给予反馈最重要的是要针对行为，而不是笼统地给一个人定性，比如“你不够专心。”反馈的内容必须是可操作的，反馈对象必须理解他们的行为需要做出哪些特定的改变。

第四章：人们的意见应始终以事实为依据

内容概要

  1. 鼓励以事实为依据的公开辩论。
  2. 不过分依赖数据，但会使用来自数据分析的洞察对团队决策进行补充。
  3. 牢记基于事实≠真实，不断地对观点进行再次审视和讨论。
  4. 辩论陷入僵局时，尝试站在对方的立场上辩论，找到自己立场中的漏洞。

观点摘录

  1. 人们必须通过探求事实来完善自己的观点，并且以开放的心态去倾听那些他们并不认同，但以事实为依据的辩论。
  2. 数据只是解决问题的一个部分，即便每一个部门的每一个人都拥有同样的数据，你依然需要让大家就业务的各个环节进行辩论，而这些是数据做不到的
  3. 在指标上犯的最大的错误之一，就是关注那些毫不重要的指标。在指标上犯的另一个错误，就是认为它们是固定不变的。指标必须是动态的，我们必须不断地重新审查和质疑这些指标，才会引出激烈的辩论。
  4. 让辩论遵守这些准则并保持文明的一个最好的办法，就是把辩论摆到一群人面前。高管们经常会把分歧藏在心里，然而这些分歧可能是那些基层员工需要理解和权衡的最重要的参考内容。

第五章：不要以为你现在的员工能够成长到承担将来的责任

内容概要

  1. 面向未来去思考你需要什么样的团队。
  2. 站在6个月后的未来审视你现在的团队，了解团队对即将到来的变化是否已准备就绪。
  3. 让每个人都理解团队需要持续“进化”。
  4. 有些人永远无法成长为未来组织中的高绩效者，主动让他们离开。
  5. 持续不断地搜寻人才，招入谁和解聘谁的决定必须完全建立在团队绩效的基础上。

观点摘录

  1. 你应该从未来的规划出发，建立一支理想的团队：确认要解决的问题、确认解决问题的时间期限、确认能够成功解决这些问题的人选，并确认要解决这些问题他们该怎么做。然后问自己，需要做哪些准备，以及需要招入什么样的人
  2. 究竟是从公司内部提拔还是从外部招聘一名高绩效者，我们的经验是看这项有待完成的工作是否需要内部人员所不具备的专业技能，或者在这项工作所属的领域，我们自己是否就身处创新前沿？
  3. 他们最应该为员工做的一件事情，就是确保公司能够生产出好产品，可以及时地服务好客户。他们并不亏欠员工一个机会让其从事自己没有准备好也没有天赋完成的工作；他们并不亏欠员工一个新设的岗位来奖励员工的付出；他们当然也没有必要为了员工而在那些可以让公司发展的人员变动上退缩。

第六章：不是每个岗位都需要爱因斯坦，但每个岗位都需要最适合的员工

内容概要

  1. 用人经理是首席招聘人员，招聘高绩效者是他最重要的工作。
  2. 招聘优秀人才不是招聘“一流选手”，而是为岗位需求找到最佳匹配人员。
  3. 确保每一个对业务至关重要的岗位上，都有一位一流人才。
  4. 主动让那些已经很出色的人离开，以便给顶尖人才腾位置。

观点摘录

  1. 你能够为员工做的最好的事情，就是只招聘那些高绩效的员工来和他们一起工作。
  2. 对于今天职场人士的最佳建议就是：保持灵活，不断学习新技能，不断考虑新机会，经常接受新挑战，这样可以保持工作的新鲜感和延展性。

第七章：无论奈飞的业绩表现是好是坏我们都按照市场顶级水平付薪

内容概要

  1. 不要依赖于薪酬调研，因为薪酬调研的信息总是滞后的。
  2. 薪酬与年度绩效评估流程无关，只与员工的绩效相关。
  3. 如果不能针对所有的岗位支付市场最高水平的薪水，优先考虑对公司的业务增长最为重要的岗位。
  4. 资历相当的应聘者应该获得同样的薪酬，跟他们之前的薪酬、他们的性别都无关。
  5. 建立薪酬透明制度，让大家对薪酬有更好的判断。

观点摘录

  1. 按市场定薪不应该是将你的薪酬水平定在整体市场范围的某个固定水平上，而应该是在你要求的工作时间范围内，估算一个人为你带来的整体市场价值。

第八章：离开的时候好好说再见，理想的公司就是那种离开之后仍然觉得它很伟大的公司

内容概要

  1. 如果员工的表现不够好，及时告诉他们要么纠正过来，要么去一家新公司。
  2. 不要把与工作不再匹配的员工归结为失败者。
  3. 不要给员工无法实现的承诺，这只会让他们感觉自己被背叛了。
  4. 积极地帮助离职员工找到新的好机会。

观点摘录

  1. 我们都应该做好准备，时不时地换一份工作，不论是在公司内部还是去一家新公司，目的就是以我们喜爱的方式工作，做那些让我们充满激情的事情。同时，假如我们的表现不够好，也应该有人告诉我们，要么快速纠正过来，要么去一家新公司。
