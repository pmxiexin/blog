直播打赏类产品特性的分析

发表于 2018-12-30 | 分类于 经验总结
字数统计: 828 | 阅读时长 ≈ 2 分钟

消费内容

打赏类业务有以下消费内容和场景

  1. 感谢型消费：
    冲动消费。用户感谢主播的辛苦付出，如唱歌。此类消费价额偏低。（小礼物走一走）
  2. 服务型消费：
    理智消费。用户为主播劳动付报酬，如“带人上分”。此类消费额中等。
  3. 自我型消费：
    冲动消费。用户为满足个人心理目标消费，如“等级”，每个用户消费达到个人身份预期。此类消费额往往偏高。
  4. 商业型消费：
    理智消费。用户为获得资源推广。此类消费额偏高。
  5. 感情（上帝/爱情）型消费：
    冲动消费。特殊的心理，可遇不可求。此类消费额极高。

对产品的帮助

  1. 由于冲动消费占主要部分，所以
    • 需尽可能简化充值流程，让用户在充值时不做思考
    • 需塑造洗脑式消费体验，让用户容易“缺乏理智”，如“连击特效”
    • 需控制付费氛围，引导用户跟风，如“免费礼物”
    • 需引导舆论氛围，引导用户多关注土豪，如“贵族发言”
  2. 根据消费内容，所以
    • 需引导用户更深层的消费意愿，如“攀比”、“报仇”、“跟风”等
    • 需提供足够的付费效果，如付费用户的曝光、特效
    • 需尽可能“消费主播”，如增加主播服务内容
    • 需增加足够的引导提示，如“主播累了，打赏一下吧”
    • 收紧商业型消费，商业用户需付出更高成本。
  3. 由于主播是消费主题，所以
  • 与主播充分配合，共同营造“付费理由”
  • 为主播提供“便捷”，如帮助主播注意高潜力付费用户（进场特效）

消费用户

直播打赏类的用户消费有以下特征

  1. 分化极大：
    极少数的用户，占到收入的绝大比重（0.5%->40%）
  2. 极不稳定：
    由于是冲动消费，用户打赏金额和情绪有关，付费额极不稳定。
  3. 有上限：
    平台粉消费远大于死忠粉。单个主播消费上限为主播能力。
  4. 缺少平台粘性&受主播影响大
    主播粉多，平台粉上。用户付费因为主播。

对产品的帮助

  1. 既需考虑付费用户，又需照顾免费用户
    • 付费用户享有一切特权，免费用户仅供体验
  2. 对不同的用户类型，需施以不同的目标，所以
    • 大R用户：极少量但影响收入最大，需提高付费额、续费率和付费上限。由专人维系。
    • 中R用户：中量用户，收入占比中等，需施以小目标鼓励更多消费。
    • 小R用户：大量用户，但是收入仅占极少，需用以提高活跃。
    • 免费用户：通过首冲活动引导付费。
  3. 需注意用户的直播转化，引导死忠粉用户为平台粉用户，所以
    • 提高高付费用户在平台的账号价值，让用户账号值钱
    • 引导死忠粉用户观看其他主播并消费
  4. 利用主播影响，让用户为平台付费
    • 用户消费需注意主播的回报，但是特权以平台为主体
