典型竞猜产品商业模式分析

发表于 2019-02-18
字数统计: 1.6k | 阅读时长 ≈ 5 分钟

概述

竞猜平台通过专业的博彩公司提供实时玩法和赔率。用户投入资金进行参与。若用户胜利和赢得相应金额，若用户失败则资金归竞猜平台所有。
由于竞猜产品有以下特征，导致其商业模式以“忠诚计划”为核心。

  1. 有专业的风控系统，在足够的量级下能保证盈利。所以行话”不怕用户赢钱，就怕用户不玩“。
  2. 付费用户基数低，只有非常少数的用户会参与到“现金”竞猜中
  3. ARPU值大，单个大玩家单笔竞猜量极大（不透露）
  4. 产品差异小，各家产品都同质化严重
    注意：竞猜在国内属于违法业务，所以主流竞猜平台都会通过“包装”的形式进行规避，属于“灰色产业”。
    更多竞猜类产品产品特征的分析可前往竞猜类产品特性的分析

过程说明

  1. 基础构建：完成最基础的竞猜功能，包括
    1. 竞猜玩法维护：通过自建操盘团队，或是购买专业玩法方案。付费额越高的用户对玩法的要求越高。大型竞猜平台往往都买自同一家博彩公司。
    2. 充值/取现：用户可进行资金的投入以及资金的提取。竞猜用户很在意资金的安全性，竞猜平台需保证充值/取现安全可靠。
  2. 用户拉新：获取用户，包括以下途径
    1. 搜索引擎：通过搜索引擎进入，但现在对搜索引擎管理越发严格，该部分流量越来越少。
    2. 广告：通过付费广告获取用户，但转化率往往都不高。
    3. 流量合作：与其他产品进行流量互换合作，需要产品匹配度高
    4. 用户推荐：通过其他用户邀请。
  3. 访客转化：通过拉新计划来让访客用户快速参与竞猜，一般有
    1. 新人活动：对于新竞猜用户的特殊优惠，基本所有竞猜平台都有（不一定通过产品的方式）
  4. 用户转化：进一步获取已竞猜的用户的价值
    1. 忠诚度计划：竞猜平台的核心。大型竞猜平台的收入往往只来源于数百个大玩家，甚至关闭新人渠道，只维护这些大玩家就能盆满钵满。大玩家一般都有专属的客服小美眉维护，任何问题都第一时间响应。小玩家也有自己的微信群或是QQ群维护，不定期举办各种活动，甚至直接发钱。
    2. 邀请计划：邀请在竞猜平台中是最主流的获取竞猜用户的方式。竞猜用户基数低，撒大网方式转化率也极低。但是竞猜用户往往都会“抱团”，并且会互相交流。大玩家的朋友也一般都是大玩家。主流的竞猜平台都会建立多级的邀请系统。竞猜平台创立初期需有一个业内资深玩家，通过其人脉即可发展成熟。
  5. 收入反馈：将已获得的收入进一步反哺平台
    1. 影响流量：购买付费流量，如SEO优化、广告等。
    2. 影响拉新计划、忠诚度计划、邀请计划：增加对各项计划的资金投入，提升效果

火猫竞猜发展阶段及关注指标

  1. 阶段零：发现需求
    1. 阶段目标：发现了可进入的市场真正面临的、未能很好满足的需求
    2. 核心指标：能够找打用户真正的痛点（即“用户玩电竞竞猜”）
    3. 阶段结果：
      1. 发现国外如bet365（国外竞猜平台）非常火爆，很多玩家参与。国内如VPgame也有很多人参与
      2. 直播时也有非常多的用户发言内容和竞猜相关
        阶段零已被行业先行者验证，后来者直接从阶段一开始
  2. 阶段一：移情阶段
    1. 阶段目标：找到了用户可接受，市场可允许的实现方式/重复他人已实现的实现方式
    2. 核心指标：找到可重复的，可规模化的实现方式
    3. 阶段结果：
      1. 找到了电竞竞猜的专业赔率公司和操盘团队
      2. 用户愿意投入资金参与到电竞竞猜中
    4. 核心功能：
      1. 竞猜玩法维护：通过购买专业玩法方案维护一套专业的竞猜玩法
      2. 充值/取现：维护一套稳定的并且可规避风险的充/取通道
  3. 阶段二：粘性阶段
    1. 阶段目标：找到了正确的产品/特性/功能来留住用户
    2. 阶段指标：用户留存率，用户在次日/7日/30日是否还会来竞猜
    3. 阶段结果：通过专业的盘口和方便的功能，提升用户留存率。
    4. 核心功能：
      1. 体验优化：优化最基础的用户参与体验
      2. 运营功能：活动公告，站内信等，能够让运营发挥能动性的功能
  4. 阶段三：病毒化阶段（进行中）
    1. 阶段目标：用户和功能自然真正或非自然地刺激了增长
    2. 阶段指标：新用户增量，每天有多少新增竞猜用户
    3. 阶段结果：
      1. 通过火猫直播导流，有大量用户进行参与转化
      2. 通过邀请制度，老用户邀请新用户参与
    4. 核心功能：
      1. 流量引导：在火猫直播内提供显著入口引导像竞猜，导入直播用户
      2. 邀请返利：方便用户进行邀请，可实时看到邀请结果
  5. 阶段四：营收阶段（进行中）
    1. 阶段目标：从健康的生态系统中获得相应利润
    2. 阶段指标：
      1. 收入额：竞猜的总收入额
      2. 付费率：新用户成为付费用户的比例，提升付费率能提升整体收入
      3. ARPU：平均每个用户的付费额，提升ARPU能提升整体收入
    3. 阶段结果：获得可观的竞猜收入
    4. 核心功能：
      1. 忠诚度计划：通过等级/积分系统维护用户
      2. 人工维护：建立快速响应，专人维护的团队
        注意：竞猜平台没有规模化阶段，毕竟是灰色产业，注重闷声发大财。
