产品助理的培养

发表于 2018-12-30 | 分类于 职业思考
字数统计: 1.4k | 阅读时长 ≈ 4

素质背景

  1. 有较强的学习能力和学习意愿，但是缺少产品职业技能的培养
  2. 有较强的沟通表达能力，但是缺少复杂工作环境下的处理经验
  3. 缺少项目经验和产品职业生涯的认知

培养内容

阶段一：流程意识（不做错事）、执行力——————意识类（习惯养成）

  1. 流程意识：
    1. 认真梳理项目流程，精确到任何典型情况，反复赠改更新
    2. 经历足够多的项目，遇到足够多的项目问题
  2. 执行力：
    1. 有足够的执行意愿，并且努力执行下去
    2. 有足够的项目管理&个人管理的技巧和能力，能使用GTD等工具
    3. 经历足够多的项目，养成良好的项目习惯，能做到定期检查和校验

阶段二：产品技能（做对事）、沟通能力——————技能类（学习&练习）

  1. 专业能力：含需求分析、文档撰写、原型图绘制、设计理解、技术理解
    1. 有着足够的学习意愿，能够快速吸收新鲜事物
    2. 与导师&其他产品经理进行技能交流，网上寻找资料，学习他人方法
    3. 体验足够多的产品，了解其他产品的设计思路和经验教训
    4. 经过足够多的，各个类型的项目，并将自己最新的技能应用其中
    5. 适时总结技能经验，并根据实际操作调整更新
  2. 沟通能力：
    1. 有足够的沟通意愿，并且努力进行下去
    2. 掌握一定的沟通技巧，如阅读数据，与导师学习等

培养原则

  1. 工作内容：允许范围内安排能使其获得成长的项目，并尽可能与个人意愿相符
  2. 工作指导：在其有独立思考后，再提供个人的建议
  3. 习惯养成：采用强制性方式使其必须养成习惯，并意识到习惯重要性
  4. 个人情感：正确的鼓励和教育，并尽量保持亦师亦友的关系

培养方式

工作

  1. 目标：
    1. 培养产品执行力，能够跟进安排的需求并无需操心
    2. 能够独立策划需求并快速输出，持续跟进不出问题
  2. 方法：
    1. 习惯养成：
      1. 个人/项目管理习惯：每天/每周上午安排今日个人/项目事项，并随着情况不断调整 → 能够合理的安排个人的工作
      2. 进度检查/周知习惯：持续检查事情推动进度 ，每天/每周下班时反馈个人/项目情况 → 能够保证进度的正常进行
      3. 基础职业习惯：及时提醒其写会议纪要、写邮件等流程习惯 → 能够避免工作中的错误
    2. 工作内容安排：
      1. 聚焦于某一具体领域，而非零碎需求 → 工作有整体感和责任感
      2. 足够多的工作项目 → 有稳定持续的成长空间
      3. 循序渐进，适当超出其能力范围的项目从只出文档，到可以出原型，到可以出方案 → 脱离舒适区，能够获得新知识并保持持续成长
      4. 前期以跟进为主，后期可有自己独立思考 → 先保证能做事，不做错事
      5. 适当的安排些非工作日常的任务 → 对工作不会倦怠
    3. 工作指导：
      1. 入职时进行一次沟通，使其明确其在公司的位置和能获得的能力 → 让其有准备
      2. 定期进行工作沟通，帮助其回顾工作内容，和碰到的问题 → 让其能够保持正确的成长
      3. 给予足够的信任和个人思考的空间 → 只有非出场不可时再出场，让其能自己多思考和沉淀
      4. 解决问题时明确指出其问题所在 → 帮助改正问题

个人

  1. 目标：
    1. 对产品永远保持着好奇心、敬畏感和兴趣
    2. 对结果极致的追求，永不放弃
    3. 能够融洽的与合作伙伴相处
    4. 保持良好的工作态度和压力处理
    5. 尽量能够保持亦师亦友的关系
  2. 方法
    1. 在与之沟通和讨论时，能够站在更大的视角去解释 → 让其保持好奇心、敬畏感
    2. 适当安排一些具备创意的需求 → 让其保持产品兴趣
    3. . 适当安排一些需要多方沟通的需求，主动帮其牵桥拉线沟通 → 沟通多了，就会学习更多的沟通技巧和情商
      1. 让其经历一些高压业务 → 经历压力多了，就会适应压力
    5. 工作中严格要求，自身也应做个好榜样 → 先被动让其必须严格要求自己，然后慢慢培养自己对产品极致的追求
    6. 平时保持严格和紧张，但是偶尔流露出来的关心，最致命

培养心得

  1. 不是所有人都适合培养，如果没有培养的条件则及时收手，无需纠缠。只需发挥其对公司价值即可。
  2. 不同人培养的目标不一样，需要补足个人短板并发挥其长处，而不是所有人都是一个模板的产出。
  3. 多沟通比什么都重要，一定要多沟通
  4. 自己要做一个有趣的人，有个人魅力能让其更喜爱
