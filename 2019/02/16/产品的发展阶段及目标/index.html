产品的发展阶段及目标

发表于 2019-02-16 | 分类于 职业技能
字数统计: 1.1k 字 | 阅读时长 ≈ 5 分钟

“产品的阶段和目标”是一个非常宏大的命题，关乎着任何一家创业公司的生死存亡，一般是老板们考虑的问题。所以我也只能根据历史和书籍知识，给大家分享一些总结。
虽然这篇文章并不能帮助产品更好战胜对手，但至少可以帮助产品看待一个产品的各个发展阶段，以及在自己从零到一负责产品时，应当注意什么事情。

精益数据分析的模型

《精益数据分析》作者发现，其实大多数产品增长理论其实殊途同归：莫瑞亚精益画布、精益创业、麦克卢尔的海岛指标（AARRR）、肖恩·埃利斯的创业增长金字塔。如下图所示：

总结下来，产品分为五个阶段，每个阶段有不同的目标，完成上一个目标后，方能进入下一个阶段。

  1. 大前提：我发现了可进入的市场真正面临的、未能很好满足的需求
  2. 移情阶段（用户调研，需求分析）：找到了用户可以接受并参与的方式
  3. 粘性阶段（留存）：找到了正确的产品/特性/功能来留住用户
  4. 病毒性阶段（拉新）：用户和功能自然真正或非自然地刺激了增长
  5. 营收阶段：找到了一个可持续发展的、可扩大规模的商业模式，并且在健康的生态系统中可获得相应利润
  6. 规模化阶段：提升效率，降低成本，利用体量产生规模化效应

以直播平台来举例

  1. 大前提：
    1. 发现Twitch在国外非常火爆，很多人愿意观看
    2. 国内游戏视频行业火热，知名电竞选手人气升高
  2. 移情阶段：
    1. 当知名玩家开直播时，很多用户愿意持续观看
    2. 看直播成为很多用户打消无聊的第一选择
    3. 很多用户自发的进行礼物打赏
  3. 粘性阶段：
    1. 通过推荐系统，让用户永远都有自己喜欢的直播可看
    2. 强化直播时的互动参与，让用户对主播粘性更高
  4. 病毒性阶段：
    1. 知名玩家id冠名，获取游戏内用户
    2. 超高薪招聘知名主播，引起社会关注
    3. 大范围社会化营销，发主播相关文章吸引热点
    4. 地铁、电影等领域的大范围广告投放
  5. 营收阶段：
    1. 从传统秀场中找到了打赏的收入方式，一切互动向秀场靠近
    2. 从体育直播中找到了直播广告的方式，开始有广告商的参与
    3. 从体育竞猜中找到了点击竞猜的方式，直播植入竞猜互动
  6. 规模化阶段：
    1. 开始有大型公会的产生，帮助培养主播，降低主播门槛和招聘成本
    2. 更精细化的活动运营，各个付费群体都有针对性的运营活动

直播平台的发展过程，也与模型所述的阶段吻合。但实际中却是多个阶段并行的，比如病毒性阶段的提前——在尚未证实直播是能持续性的时候，就大范围的开始投入广告。导致如映客等平台在热潮退去后量级无法持续，估值不断下跌。因为为了获得资本青睐，需要尽快抢占第一直播平台，即使用户留存率低也需要更大量的新用户涌入。

总体来说，在资本促使下，风口上的产品都难以按照自己的既定轨道发展，各个阶段都被压缩。特别是病毒性阶段，贯穿着风口产品的全部周期，需要的是尽快的抢占先机，获得资本青睐。
但对于小而美的产品来说，依然可以通过这五个阶段以及不同的目标设定，来帮助自己稳扎稳打的发展产品。所以当前的产品目标设置成什么，就是看自己目前处于哪个阶段。
