典型直播产品商业模式分析

发表于 2019-02-17 | 分类于 经验总结
字数统计: 2.5k 字 | 阅读时长 ≈ 12 分钟

概述

主播在直播平台上进行直播，用户以礼物的形式进行打赏，打赏收入由主播和直播平台进行分成，是典型的双边市场。
由于直播产品具备着“流量大，同时在线人数高，但付费率低”的特征，所以商业模式以“流量转化”为核心。
直播产品有三个参与者，用户、主播和平台，主要的参与方式如下

  1. 用户：观看直播，如令其满意可进行付费打赏。
  2. 主播：通过直播内容获得观众，并吸引主播打赏
  3. 平台：负责主播招募和基础功能搭建，以及用户的引入转化

以传统的“酒楼表演”来举例：用户是看客，看得开心了就打赏点钱； 主播是表演者，表演得好有更多人来看；平台是酒楼，招募表演者，也提供表演场地和瓜子板凳。

过程说明

  1. 主播招募：获得直播内容，一般有这些内容：
    1. 职业主播：与个人主播或公会合作，以“签约”的形式招募主播，提供“基础薪酬”和“打赏分成”；
    2. 非职业主播：部分普通用户希望赚点零花钱，自发开播。只有打赏分成收入，没有基础薪资，如效果好可能会被公会和平台招募；
    3. 合作内容：如“DOTA2赛事直播”、“暴走大事件”视频轮播、“小米发布会”直播等，以商务合作形式获得直播内容。
  2. 流量引入：有最基本的内容后，需要引入观众，获得收入，有那么几种渠道
    1. 回访用户：留存的观看用户，需关注留存率
    2. 搜索：通过搜索引擎来的用户，需关注SEO优化情况
    3. 应用市场：通过应用市场搜索来的用户，需关注ASO优化情况
    4. 广告：通过付费广告获取的用户，需关注付费成本
    5. 流量合作：与合作方进行流量合作，进行资源互换，需关注合作渠道与自身平台的匹配度
    6. 用户推荐：忠实用户向他人推荐获取的新用户，需关注用户口碑
  3. 访客流量转化：当用户来到平台后，最终需要变现。直接通过广告变现或是间接通过观看后的付费打赏变现，主要有那么几种流量转化方式
    1. 搜索渠道：用户通过站内搜索，到达直播间。一般都是“主播粉”和“赛事粉”用户，有明确的观看目的
    2. 各级导航：用户通过站内导航，到达直播间。一般都有明确的观看习惯，比如看美女主播，看DOTA2主播，明星主播等
    3. 喜好推荐：用户根据平台推荐，到达直播间。一般分为运营主动推荐和喜好分析推荐。
    4. 【直接变现】广告计划：通过在页面各处埋有广告，可获得最基本的广告收入
    5. 【直接变现】其他方式：其他通用的，能进行流量变现的方式，如竞猜小游戏、抓娃娃、游戏联运等
  4. 观众流量转化：用户进入直播间后，会一直停留在直播间观看，此时需要进行进一步的流量转化，有那么几种转化方向
    1. 忠诚度计划：将用户转化为忠实观众，持续性的访问平台，稳定贡献广告收入，也是付费打赏的潜在用户，一般通过等级、积分、奖励等方式刺激
    2. 【直接变现】付费计划：将用户转化为付费用户，获得打赏收入。并将付费用户进一步提升为续费用户，获得二次打赏收入。主要依靠主播自身的吸金能力，其次平台需要提供礼物道具、特权等方式刺激
    3. 分享计划：将用户转化为“分享用户”，能够将平台推荐给他人，从而进行低成本拉新。
  5. 收入反馈：以上过程已从流量来源到付费转化，下一步需将获得地收入进一步投入以实现规模化，一般有以下方式
    1. 补贴主播：通过调整主播基础薪酬、提供额外奖励的方式，使得更多的主播能够入驻直播平台，有更多的直播内容
    2. 培训主播：对主播进行技能培训，提升主播创造直播内容和吸金的能力
    3. 购买流量：将获得的收入投入SEO优化、ASO优化、广告、流量合作等方式，以获得更多的用户。

火猫直播发展阶段及关注指标

  1. 阶段零：发现需求
    1. 阶段目标：发现了可进入的市场真正面临的、未能很好满足的需求
    2. 核心指标：能够找打用户真正的痛点（即“用户想看游戏直播”）
    3. 阶段结果：
      1. 发现Twitch（国外游戏直播平台）非常火爆，很多人愿意观看
      2. 行业内还很少有DOTA2、CSGO相关的直播网站
      3. 国内游戏视频行业火热，知名电竞选手人气升高
        阶段零已被行业先行者验证，后来者直接从阶段一开始
  2. 阶段一：移情阶段
    1. 阶段目标：找到了用户可接受，市场可允许的实现方式/重复他人已实现的实现方式
    2. 核心指标：找到可重复的，可规模化的实现方式
    3. 阶段结果：
      1. 发现了“签约主播、用户观看“的实现方式
      2. 当知名玩家开直播时，很多用户愿意持续观看
      3. 看直播成为很多用户打消无聊的第一选择
    4. 核心功能：
      1. 页面基础：能够将各个渠道来的用户成功的引导到直播间
      2. 推流&直播系统：能够将主播直播画面即时的显示给用户看
  3. 阶段二：粘性阶段
    1. 阶段目标：找到了正确的产品/特性/功能来留住用户
    2. 阶段指标：
      1. 用户留存率，用户在次日/7日/30日是否还会回到直播平台
      2. 日均观看数：每天每个用户会观看多少个主播
    3. 阶段结果：
      1. 强化直播时的互动参与，让用户对主播粘性更高
      2. 通过推荐系统，让用户永远都有自己喜欢的直播可看
      3. 通过用户忠诚度计划，以等级&积分，消息推送方式留存用户
    4. 核心功能：
      1. 直播互动：用户和主播能够通过互动，用户更快成为主播粉
      2. 喜好推荐：让用户能够每天流量更多主播，提高主播利用率
      3. 消息推送：通过消息推送，提醒用户观看直播
      4. 用户激励：能够通过等级、积分等方式维系用户停留
      5. 体验优化：优化最基础的导流/观看体验
  4. 阶段三：病毒化阶段
    1. 阶段目标：用户和功能自然真正或非自然地刺激了增长
    2. 阶段指标：
      1. 新用户增量：每天有多少新用户来到平台
      2. 日均开播数：每天有多少的主播进行直播
    3. 阶段结果：
      1. 知名玩家id冠名，获取游戏内用户
      2. 确定产品差异化策略，以赛事直播作为主打，吸引赛事用户
      3. 高薪招聘知名主播，引起社会关注
    4. 核心功能：
      1. 容载扩大：提升产品能够支撑的容载量，避免因用户涌入而崩溃
      2. 赛事体系：产品差异化策略的的功能，包括赛事中心、赛事专题等
      3. 主播申请/主播后台：方便更多的主播入驻，提高主播工作满意度
  5. 阶段四：营收阶段（进行中）
    1. 阶段目标：从健康的生态系统中获得相应利润
    2. 阶段指标：
      1. 收入额，通过各种营收渠道获得的收入量总和
      2. 付费率：新用户成为付费用户的比例，提升付费率能提升整体收入
      3. ARPU：平均每个用户的付费额，提升ARPU能提升整体收入
    3. 阶段结果：
      1. 从传统秀场中找到了打赏的收入方式，一切互动向秀场靠近
      2. 从体育直播中找到了直播广告的方式，开始有广告商的参与
      3. 从体育竞猜中找到了点击竞猜的方式，直播植入竞猜互动
    4. 核心功能：
      1. 礼物打赏：用户充值后，送礼物打赏给主播的体验
      2. 广告布局：在合适的位置布局广告，获得广告收入
      3. 电竞竞猜：通过电竞竞猜，从赛事观看用户中获得收益
  6. 阶段五：规模化阶段（进行中）
    1. 阶段目标：提升效率，降低成本，利用体量产生规模化效应
    2. 阶段指标：
      1. 获客成本：统计各个渠道来源用户的成本及转化率，减少拉新成本
      2. 付费用户终身价值：提升一个付费用户在平台上累计创造的价值
      3. 主播绩效：主播绩效考核，淘汰绩效差的主播
    3. 阶段结果：
      1. 更精细化的活动运营，各个付费群体都有针对性的运营活动
      2. 开始有公会的产生，帮助培养主播，降低主播门槛和招聘成本
    4. 核心功能：
      1. 统计上报与展现后台：为精益化的数据分析提供基础
      2. 平台活动：进行用户的精细化活动运营，提升收入指标
      3. 迭代优化：优化历史遗留问题，提升各处体验
      4. 主播绩效管理后台：查看和管理主播的绩效，更好的运营主播
