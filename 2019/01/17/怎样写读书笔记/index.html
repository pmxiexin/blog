怎样写读书笔记

发表于 2019-01-17 | 分类于 读书笔记
字数统计: 1.6k 字 | 阅读时长 ≈ 7 分钟

我一直比较喜欢读书，也读了很多书。
但对于读书笔记，或是懒于写，或只是摘录名段子。
但是过了一两个月回头看，却难以回忆起当时的感悟。似乎觉得书又白读了。

现在开始认真的想一想这个问题，我为啥要读书，都是读哪些书，以及觉得怎么样写读书笔记更符合个人的习惯和要求。

为什么要读书

我读书有三个目的（目的性好强啊O__O “…）：

  1. 【P0】丰富和强化自己的思维方式，因为：
    1. 每个人所处的环境和人生经历，产生了每个人的思维。所以了任何人的思维都是有限的。比如我很难想象销售的思维，但是读书能让我感受到些许
    2. 职业需要我去理解他人眼中世界的样子。比如简单的一张海报，在设计师眼里是点线面的组合和变化，在媒介眼里是一个个文字的组合和语义，在商务眼里是背后的商业原理和商业机会……这是我好奇的
  2. 【P1】获得基本的知识点，能够在恰当的时候有意识的使用，因为：
    1. 积累知识点就像积累素材一般，能提早准备能更好的理解和使用。指不准哪天就用到了。
    2. 知识点能让我更好地分辨是非对错，让我的思考有所依据。
  3. 【P2】陶冶情操，因为：
    1. 我发现大多数的卓越的产品和产品经理，并非通过理性推导出来的。逻辑思维创造不出有趣的产品。（比如云音乐的师母和微信的张小龙都很爱引用王阳明，也不知道是不是“电梯成功论”）
    2. 大家都习惯有趣的人

我会读哪些书

  1. 思维类书籍：帮助我重新认识世界、他人和自己，比如《学会提问》、《高效能人士的七个习惯》、《失控》等
  2. 知识类数据：帮助我积累基础的知识信息，在实际工作中又能直接的应用。又分为经济类、管理类、设计类、产品类、技术类、沟通类等，比如《蓝海战略》、《联盟》、《设计心理学》、《结网》、《计算机基础》……
  3. 传记&小说类：帮助我体会我无法体会的经历，让我体味世态炎凉，比如《一网打尽》、《三体》等

读书笔记的标准和格式

根据不同的书籍类型，以及我所期望收获的内容有所不同：

  1. 思维类：
    1. 目标：能够让我经过很久后，阅读到读书笔记，依然能感受到作者的思路
    2. 阅读方式：
      1. 概览：简单浏览目录和正文内容，先有个基础了解。作者想说啥，分别从哪几个角度去说明和论证。
      2. 通读：通篇阅读全文，并在经典语句之处留下标记。
      3. 复读：完整地重新阅读一遍，详细思考作者想传递的内容，并完成对本书的心得感悟
      4. 跳读：根据标记，只阅读最重点的内容，并概述本书内容框架，以思维导图的形式呈现
      5. 摘录：根据历史阅读标记，摘录经典语句
      6. 看他人评论：阅读他人对本书的看法和读书笔记，和自己的内容做对比，如有遗漏则重新再读该部分内容。如有好的文章则链接贴在读书笔记底部
    3. 笔记内容：
      1. 书本概述：作者的核心观点，用一句话表达
      2. 【重点】读书感悟：读这本书自己的想法和感受，包括对本书的理解和不足之处的探讨
      3. 内容说明：本书的思维导图，或是其他形式，能够详述本书内容
      4. 经典摘录：摘录本书的好词好句，或是会引人深省的句子
      5. 网络好文：网络上对这本较好的笔记和感悟
  2. 知识类：
    1. 目标：能够罗列本书重要的知识点内容，很久之后能够查询知识点内容
    2. 阅读方式：
      1. 概览：简单浏览目录和正文内容，先有个基础了解。分别有哪些知识点，各个知识点之间有什么用的关系。
      2. 通读：通篇阅读全文，并对知识点的重点描述作标记。
      3. 复读：完整地重新阅读一遍，主要回顾知识点的框架和他们之间的关联关系，并完成对本书的心得感悟
      4. 跳读：只阅读知识点标记，并概述本书内容框架，以思维导图的形式呈现，并摘录知识点的重要描述
      5. 看他人评论：阅读他人对本书的看法和读书笔记，和自己的内容做对比，如有遗漏则重新再读该部分内容。如有好的文章则链接贴在读书笔记底部
    3. 笔记内容：
      1. 书本概述：本书主要描述的内容，和重要的知识点
      2. 读书感悟：读这本书自己的想法和感受，知识点可能的运用方式
      3. 【重点】内容说明：本书的思维导图，或是其他形式，能够详述本书知识点内容，需对重要描述语句作摘录。
      4. 网络好文：网络上对这本较好的笔记和感悟
  3. 传记&小说类：
    1. 目标：能够感受主人公的经历和情感；能够回忆一些主人公的经历和语录，为后续装逼
    2. 阅读方式：
      1. 通读：通篇阅读全文，并对兴趣点&重大的转折点作标记。
      2. 跳读：阅读标记点，并完成本书的新的感悟，摘录重点的语句和经历
      3. 看他人评论：阅读他人对本书的看法。如有好的文章则链接贴在读书笔记底部
    3. 笔记内容：
      1. 书本概述：本书主要描述的故事
      2. 【重点】读书感悟：读这本书自己的想法和感受，联想自己的生活
      3. 景点摘录：摘录重要的情节和语录，为后续装逼使用。
      4. 网络好文：网络上对这本较好的笔记和感悟
