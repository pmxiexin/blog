《谈判课》读书笔记

发表于 2018-12-30 | 分类于 读书笔记
字数统计: 1.3k 字 | 阅读时长 ≈ 6 分钟

一句话描述

教你公平的谈判策略让你在生活&事业中坚持使用以提高胜利概率
原则：争取更多
前提：双方都在公平的环境下谈判，双方都理智

谈判策略

  1. 目标至上：在大目标的前提下不计较是非对错和历史问题
  2. 重视对手：给予对手足够的尊重和了解
  3. 进行情感投资：让对方处于对自己有利的情绪状态
  4. 谈判形式千差万别：谈判需因地制宜
  5. 循序渐进：不要一次提太多要求（从简单的开始）
  6. 交换不等价物：用对方所重视的交换己方所不重视的
  7. 摸清对方谈判准则：对方说过什么，是否有过先例
  8. 开诚布公并积极推动谈判：不要欺骗，要足够信任
  9. 始终和对方保持沟通顺畅，之处显而易见的问题，将对方引至己方设定的道路
  10. 找出问题的症结所在并将它转变成机会：换位思考，找出真正的问题
  11. 接受双方的差异：按对方的习惯和状态进行
  12. 做好准备——谈判准备清单：做充足的准备

谈判的层次

  1. 迫使对方按照你的意愿形式：利用权力压制（反对）
  2. 让对方按照你的思路思考：基于利益的谈判
  3. 让对方理解你想让他们理解的观念：让对方理解你的想法，并遵照你的思路
  4. 让对方感受你想让他们感受的：让对方感受，自己形成观点和思路

谈判场景与技巧

对方处于情绪化过程中

  1. 及时的中断谈判，明确的指出当前正在情绪化中，不宜谈判
  2. 让双方都共同冷静一下后再谈判
  3. 换一个人进行谈判

谈判复杂的问题

  1. 先作简单寒暄，达成一些谈判项目外的共识，如天气，日常等
  2. 先提及简单的小目标，再涉及大目标
  3. 谈判过程中，及时的总结让对方反复确认
  4. 及时纪要，避免是否大家不认账

谈判艰难的问题

  1. 提早指出本次问题艰难，希望大家能保持理智
  2. 分析对方的难点并及时指出
  3. 仔细倾听，对方会提供办法（暗示性内容：如果XXX，是否就可以达成）
  4. 适当的控场和休息以保证氛围和谐
  5. 利用不等价物交换，对方在意而己方不重视的东西
  6. 持续性的沟通，坚持不懈

不熟悉或有隔阂

  1. 询问合适的第三方经验
  2. 让双方所共同信任的第三方牵头和见证

对方做错事情

  1. 让对方做出承诺或制定准则
  2. 直接指出对方问题，使其自相矛盾

自己并不在理

  1. 关注决策者，进行情感投资，给予对方足够的尊重和关怀
  2. 是否有过这样的先例
  3. 进行不等价物交换

十分有利于自己

  1. 让对方稍有收获，出让一部分利益

信息不充裕

  1. 循序渐进，询问对方信息
  2. 不要轻易承诺，直到有足够信息

谈判清单

第一象限：问题和目标

  1. 目标-长期/短期
  2. 问题：妨碍目标实现的问题有哪些？
  3. 谈判各方：决策者、对方和第三方
  4. 交易失败怎么办？最糟糕的情况是什么？
  5. 准备工作：时间、相关准备、谁掌握了更多信息？

第二象限：形势分析

  1. 需求/利益：双方的、理性的、情感上的、共同的、相互冲突的、价值不等的
  2. 观念：谈判各方脑海中的想法、角色的转换、文化、矛盾冲突、信任
  3. 沟通：风格、关系？
  4. 准则：对方的准则、谈判规范
  5. 再次检查去目标：就双方而言、为什么同意、为什么拒绝？

第三象限：选择方案/降低风险

  1. 集思广益：可以实现目标的选择有哪些，交易条件是什么？有何关联
  2. 循序渐进策略：降低风险的步骤
  3. 第三方：共同的敌人且有影响力的人
  4. 表达方式：为对方勾画蓝图、抛出问题
  5. 备选方案：如有必要对谈判进行适当调整或施加影响

第四象限：采取行动

  1. 最佳方案：破环谈判的因素、谈判中的欺诈因素
  2. 谈判发言人：发言方式和发言对象
  3. 谈判过程：议程、截至日期、时间管理
  4. 承诺/动机：主要针对对方

书籍的不足之处

  1. 并非所有谈判都要公平
    如若是一次性交道而后续无所交际，适当的“哄骗”
  2. 并非所有谈判都要和谐
    吵架自有吵架的意义，吵架是自发性引导观点的重要方式
  3. 并非所有对方都是理性
    执念的难易更改

对产品经理的意义

  1. 在工作中保持对结果的追求——争取更多
  2. 恰当的利用谈判规则与各方进行谈判
  3. 行动胜过一切
