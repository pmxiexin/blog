《高效能人士的七个习惯》读书笔记

发表于 2019-01-12 | 分类于 读书笔记
字数统计: 2.2k 字 | 阅读时长 ≈ 11 分钟

读书感悟

刚开始接触到这本书，并没提起什么兴趣，毕竟书名太格式化了，和现在泛滥的鸡汤文和成功学没什么两样，比如《成功产品的7个诀窍》，《升职加薪的7个方法》之类的。
在打开书之前简单的猜想了一下，可能作者会觉得是哪7个习惯呢？好看看我和作者之间有多大默契。结果7个里面，猜中了4个，顿时觉得这本书并没什么特殊之处啊。

但过了很久，无聊间浏览这本书的时候，发现其实我有很多潜移默化的意识和作者有很深的共鸣，但是他的理解层次却比我深刻得多，比如

  1. 我们都认为“个人长期以来的习惯决定了他每一次选择，和最终的成就”。
  2. 我们都认为“世界是有规律可循的，任何现象（比如成功）都是可拆解的”
    我只是有这个潜意识，自己也说不上来为什么会有这个想法。但作者会详细地，有条理的慢慢道来，为什么他觉得这个是对的。
    于是我开始认真的对待这本书。
    ——————————————————————————————
    读完以后，还是发现自己读成功学的书的确没啥用。。。
    这些好的习惯，从小到大被家庭和学校灌输着，哪怕问社会中最失意的人，什么习惯能让你获得成功，每个人都能说得头头是道。但正如阿尔·帕西诺在《闻香识女人》中所说：

    现在我到了一个人生的十字路口，我一向知道哪条路是正确的，这毋庸置疑。我知道，可我没走，为什么？因为这太他妈难了。
    但这并非是最可怕的事情，如果一个人能意识到能有对错意识，这人至少尚存良知，还有一丝能靠理性和毅力走到正确的路途的希望。即使走上了错误的路也自愿承担着后果，明明白白地看着自己堕落。
    更可怕的是，大多数人（包括我）根本无法意识到自己正面临选择，无法意识到自己走上了错误的路，浑浑噩噩地走向终点，发现是条死路。但往回看时，依然发现还是只有一条路。于是责怪命运的安排。

所以这本书的确阐述了七个好习惯，解释的很清楚。
我也相信能意识到这七个习惯的，至少是个有良知的人；而能执行七个习惯的，都能获得一定的成功。
可惜大多数人并不会捧起这本书。能捧起这本书的人都有个习惯——“内省”

内省是保持自我的良知，纠正习惯的前提。
每天早上和晚上，每当做任何决策之前，都停一停。默念一遍七个习惯，并想象自己站在了十字路口，分辨哪一条是正确的，哪一条是错误的。并做出自己的判断。
这是我觉得高效能人士的第〇个习惯。

书本内容

这本书主要讲了三个目标，并从中贯穿了七个习惯：

  1. 个人领域的成功，从依赖到独立：意愿上积极主动（习惯一），目标上以终为时（习惯二），执行上要事第一（习惯三）
  2. 公众领域的成功，从独立到互赖：个人领域习惯的基础上，目标上双赢思维（习惯四），执行上知己知彼（习惯五）、统合综效（习惯六）
  3. （不追求成功了）自我提升和完善：意愿上不断更新（习惯七）

七个习惯分别是：

  1. 习惯一：积极主动(Be Proactive)
    积极主动即采取主动，为自己过去，现在及未来的行为负责，并依据原则及价值观，而非情绪或外在的环境来下决定。积极主动的人是改变的催生者，他们摒弃被动的受害者角色，发挥了人类四项独特的禀赋—-自觉，良知，想象力和自主意志，同时以由内而外的方式来创造改变，积极面队一切，他们选择创造自己的生命，这也是每个人最基本的决定。
  2. 习惯二：以终为始(Begin with the end in Mind)
    所有事物都经过两次的创造—– 先是在脑海里酝酿，其次才是实质的创造。个人，家庭，团队和组织在做任何计划是，均先拟出愿景和目标，并据此塑造未来，全心投注自己最重视的原则，价值观，关系及目标。对个人，家庭或组织而言，使命宣言可说是愿景的最好形式，它是主要的决策，主宰了所有其他的决定。领导工作的核心，就是基于共有的使命，愿景和价值观，创造出一个文化。
  3. 习惯三：要事第一(Put Frist Thing First)
    要事第一即实质的创造，是梦想（你的目标，愿景，价值观及要事处理顺序）的组织和实践。次要的事不必摆在第一，要事也不能放在第二。无论迫切性如何，个人和组织均要更多聚焦要事，重点是，把要事放在第一位。
  4. 习惯四：双赢思维(Think Win/Win)
    双赢思维是一种基于互敬，寻求互惠的思考框架与心意，目的是分享更多的机会，财富及资源，而非敌对式竞争。双赢既非损人利己（赢输），亦非损己利人（输赢）。我们的工作伙伴及家庭成员要从互赖式的角度来思考(‘我们’而 非’我’)。双赢思维鼓励我们解决问题，并协助个人找到互惠的解决办法，是一种资讯，力量，认可及报酬的分享。
  5. 习惯五：知彼解己 (Seek Frist To Understand，Then to be Understood)
    当我们不再急切回答，改以诚信去了解、聆听别人，便能开启真正的沟通，增进彼此的关系。对方获得了解后，会觉得受到尊重与认可，进而卸下心防，坦然而谈，双方对彼此的了解也就更顺畅自然。知彼需要仁慈心；解己需要勇气，能平衡两者，则可大幅提升沟通的效率。
  6. 习惯六：统和综效(Synergize)
    统和综效谈的是创造第三种选择——即非按照我的方式，亦非遵循你的方式，而是第三种远胜个人之见的办法。它是互相尊重的成果——不但是了解彼此，甚至是称 许彼此的差异，欣赏对方解决问题及掌握机会的手法。实现1+1>2的成果。
  7. 习惯七：不断更新(Sharpen the Saw)
    不断更新谈到的是，如何在四个基本生活面向（生理，社会/情感，心智及心灵）中，不断更新自己。这个习惯提升了其他六个习惯的实施效率。

金句摘录

  1. 领导力就是清晰地之处别人的价值和潜力，使对方受到鼓舞而有所意识。
  2. 习惯是“只是”、“技巧”与“意愿”相互交织的结果。知识是理论的范畴，指点“做什么”及“为何做”；技巧告诉我们“如何做”；意愿促使“想要做”。要养成一种习惯，三者缺一不可。
  3. 成功，甚至求生的关键并不在于你流了多少血汗，而在于你努力的方向是否正确。
  4. 四种自传式回应（听别人讲话时总会联想到自己的经历）
    1. 价值判断——对旁人的意见只有接受或不接受
    2. 追根究底——依自己的价值观探查别人的隐私
    3. 好为人师——以自己的经验提供忠告
    4. 自以为是——根据自己的行为与动机衡量别人的行为与动机
  5. 成熟模式图：人类成长的三个极端
    1. 【依赖期】以你为核心：你照顾我；你为我的得失成败负责
    2. 【独立期】以我为核心：我可以做到；我可以负责；我可以靠自己；我有权选择；
    3. 【互赖期】以我们会核心：我们可以做到；我们可以合作；我们可以融合彼此的智慧和能力，共创前程；
  6. 效能包含两个要素：一是产出，二是产能——生产的资质和能力
  7. 责任性授权要求双方就五个方面达成清晰、坦诚的共识，并做出承诺：预期成果、指导方针、可用资源、责任归属、明确奖惩
  8. 情感账户的七种投资方式：理解他人、注意小节、信守承诺、明确期望、正直诚信、勇于致歉、无条件的爱
