如何招聘面试产品经理

发表于 2018-12-30 | 分类于 职业思考
字数统计: 1.7k | 阅读时长 ≈ 5 分钟

产品实习生（短期实习）

  1. 沟通能力
    1. 评判标准：
      1. 是否能自信的、从容不迫的表达个人观点
      2. 是否可以快速理解他人观点
    2. 面试题：
      1. 自我介绍（自信、结构化）
      2. 为什么选择产品经理/火猫直播/转行
      3. 这次实习有明确的目标么？想要了解什么，学到什么？
      4. 如果你的意见和导师的意见不和怎么办
  2. 细心&认真
    1. 评判标准：
      1. 能否做好个人管理，细心做事
      2. 是否能做到充分沟通并遵循导师意见
    2. 面试题：
      1. 你有养成过什么习惯么？怎么完成的
      2. 你做过什么事让你觉得最认真，怎么完成的
  3. 执行力
    1. 评判标准：
      1. 是否以结果为导向
      2. 是否对个人信念永不放弃
    2. 面试题：
      1. 能举例一个认为自己执行力很强的例子么
      2. 如果项目延期了怎么办（复杂状况）

产品应届生

  1. 沟通能力
    1. 评判标准：
      1. 是否能自信的、从容不迫的表达个人观点
      2. 是否表达观点可以足够简明扼要和结构化输出
      3. 是否可以快速理解他人观点
      4. 复杂状况下能否快速处理沟通关系
    2. 面试题：
      1. 自我介绍（自信、结构化）
      2. 为什么选择产品经理/火猫直播/转行（简明扼要）
      3. 介绍一下最喜欢的App/项目（简明扼要，结构化）
      4. 如果项目延期了怎么办（复杂状况）
  2. 学习能力
    1. 评判标准：
      1. 是否有足够的学习意愿和自我驱动力
      2. 是否掌握了学习技巧和较高的学习效率
    2. 面试题：
      1. 为了产品经理都做了什么准备（学习意愿）
      2. XXX书籍讲述的主要是什么内容（学习技巧）
  3. 产品直觉
    1. 评判标准：
      1. 是否对产品有好奇心，体验足够多的产品
      2. 是否注意平时观察产品细节
      3. 是否对互联网行业深度热爱，并能独立思考
    2. 面试题：
      1. 有什么比较冷门的，但是你很喜欢的App可以推荐一下么（好奇心）
      2. 如果再让你做一个功能，你打算做什么（产品细节）
      3. 你怎么看待DOTA2/王者荣耀这款游戏（行业思考）
  4. 执行力
    1. 评判标准：
      1. 是否以结果为导向
      2. 是否对个人信念永不放弃
    2. 面试题：
      1. 能举例一个认为自己执行力很强的例子么
      2. 如果项目延期了怎么办（复杂状况）
  5. 逻辑能力
    1. 评判标准：
      1. 能进行结构化的思考和结构化表达
      2. 能进行深度思考，归纳和演绎
    2. 面试题：
      1. 25匹马，5个跑道，每个跑道最多能有1匹马进行比赛，最少比多少次能比出前3名？（深度思考）
      2. 你觉得自己为什么适合产品经理岗位？

产品经理（1~3年）

  1. 沟通能力
    1. 评判标准：
      1. 是否能自信的、从容不迫的表达个人观点
      2. 是否表达观点可以足够简明扼要和结构化输出
      3. 是否可以快速理解他人观点
      4. 复杂状况下能否快速处理沟通关系
      5. 是否具备足够的情商，能处理复杂的人际关系
    2. 面试题：
      1. 自我介绍（自信、结构化）
      2. 为什么从上一家公司离职/选择火猫直播（简明扼要）
      3. 介绍一下你最成功&失败的App/项目（简明扼要，结构化）
      4. 如果项目延期了怎么办（复杂状况）
      5. 如果你的意见和老板不合？打算怎么处理？（情商）
  2. 学习能力
    1. 评判标准：
      1. 是否有足够的学习意愿和自我驱动力
      2. 是否掌握了学习技巧和较高的学习效率
      3. 是否足够了解产品经理职业规划
      4. 是否有做经验沉淀和总结
    2. 面试题：
      1. 最近都有看什么书吗？都讲述的是什么内容（学习意愿）
      2. 产品经理怎么样保持持续进步呢（学习技巧）
      3. 你认为产品经理分为哪几个职业阶段呢（职业规划）
      4. 有什么产品经理工作经验（如撰写文档、绘制原型）可以分享吗（沉淀&总结）
  3. 产品直觉
    1. 评判标准：
      1. 是否对产品有好奇心，体验足够多的产品
      2. 是否注意平时观察产品细节
      3. 是否对互联网行业深度热爱，并能独立思考
    2. 面试题：
      1. 有什么比较冷门的，但是你很喜欢的App可以推荐一下么（好奇心）
      2. 如果再让你做一个功能，你打算做什么（产品细节）
      3. 怎么看待你所处行业的发展呢？最后会变成什么样子（行业思考）
      4. 你觉得你所在的公司，还有什么机会可以生存/发展（行业思考）
      5. 你怎么看待DOTA2/王者荣耀这款游戏（行业思考）
  4. 执行力
    1. 评判标准：
      1. 是否以结果为导向
      2. 是否对个人信念永不放弃
      3. 具备基础的项目管理知识，能够做好个人和项目的管理
    2. 面试题：
      1. 能举例一个认为自己执行力很强的例子么
      2. 如果项目延期了怎么办（永不放弃）
      3. 你的产品当前存在着什么样的问题？为什么容忍了（永不放弃）
      4. 你怎么理解项目管理？有具体的推进项目的技巧可以分享吗？（项目管理）
  5. 逻辑能力
    1. 评判标准：
      1. 能进行结构化的思考和结构化表达
      2. 能进行深度思考，归纳和演绎
    2. 面试题：
      1. 25匹马，5个跑道，每个跑道最多能有1匹马进行比赛，最少比多少次能比出前3名？（深度思考）
      2. 你觉得自己为什么适合产品经理岗位？
  6. 数据分析
    1. 评判标准：
      1. 有基本的数据意识
      2. 有通过数据去寻找原因的经历和方法论
    2. 面试题：
      1. 假设数据增长你会觉得是什么原因。怎么找出原因所在？
      2. 你的核心KPI是什么？为什么确定的是这个。如果再让你做一个提高KPI的工呢过，打算做什么？
  7. 工作态度
    1. 评判标准：
      1. 认真对待工作，不懈怠
      2. 在基础工作的基础上，能够有所创新&思考
    2. 面试题：
      1. 能解释一下XX词语是什么意思吗？如，解释一下“带宽”（认真、细节）
      2. 描述一下你的产品逻辑，能解释一下为什么是XX而不是XX吗？如解释一下为什么排行榜是前10名而不是前20名。（有自己的思考）
