《信息设计》读书笔记

发表于 2016-12-30 | 分类于 读书笔记
字数统计: 1.8k 字 | 阅读时长 ≈ 9 分钟

概述
信息设计是将获取的信息为信息再次被获取而表达的过程

概念解释

  • 外部世界：客观事物的信息。如物品的颜色、他人的动作、社会现象、时间的流逝、力量的大小、位置、工资表等。
  • 感官：将外部世界转化为内部世界信息的途径，主要是五感。
  • 内部世界：头脑中的信息。如红色代表警示等。人脑内部的是以符号、符号结构以及符号操作来表征、解释外部世界的。“符号”是信息的载体，因此这些心理表征就代表了外部世界存贮在头脑中的信息。——诺曼
  • 信息获取：外部世界的信息通过感官转化为内部信息的过程，当外部世界的信息与内部世界的信息相符时，外部的信息能顺利的转化为内部的信息。当外部信息与内部信息冲突时，外部的信息难于被接受，并容易出错，必须产生新的符号以解释外部的信息使得外部的信息能被接受。信息获取是为了信息被表达。
  • 设计表达：设计师通过其设计模型将内部世界的信息进行加工并通过五感的途径转化为外部世界信息（系统表象）的过程。当产生的外部世界的信息与受众的内部世界符号（用户模型）相符时，设计师的内部世界的信息能顺利准确地被受众获取。一种可取的方式是利用自然匹配与社会语义符号，因为这两种符号也是大众都拥有的符号。设计表达是为了信息被受众获取。
  • 信息设计：信息设计的完整过程是设计师通过感官获取外部世界的信息，其加工转化为内部世界的信息，并通过感官设计将其内部世界的信息转化为外部世界信息并被用户获取。信息设计的目的是为了该过程的信息尽可能顺利并转化率高。

例子

任何信息设计都有“ 外部世界->感官->内部世界->感官->外部世界 ”的过程

如中国国旗。中国历史及国情（内部世界） -> 五感 -> 设计师对国家的情感(内部世界) -> 视觉 -> 中国国旗（外部世界）
国旗红色代表激昂与鲜血，黄色星星代表党。来源是历史原因和生理原因。

如音频图谱。音乐节奏（外部世界） -> 听觉 ->节奏感（内部世界） -> 视觉 -> 音频图谱（外部世界）。
图谱柱状图的柱高低代表声音响度，来源可能是生活中堆积的高度代表多少。

如触摸屏上的一个发送按钮。现实生活中的按钮(外部世界) -> 视觉和触觉 -> 按钮的头脑符号（内部世界） -> 视觉 -> 屏幕上的按钮（外部世界）
你会想按它是因为它符合你生活中对按钮的认知。

静态信息、动态信息与交互信息的维度

静态信息和动态信息是单向的，即其中的两个“外部世界”是不同的，一般进行信息设计的时候会再增加维度以利于理解。，即从高维看低维的信息会更加清晰，即整体俯视。
如2008奖牌数的表格是国家和奖牌两个维度，而做成条状图即多了一个数量维度。数字原是零维信息，现在有了方向成为了一维信息。

如东莞迁徙。右侧的表格是两个维度，热度和地点。但是在图像里成为了三个维度，将一维的地点维度变为二维的地图维度。而热度用亮度一个维度表现。

如拿破仑征战俄国图，数据本身是四个维度，地点，时间，人数，温度。现在将地点用二维地图表示，时间用一维方向（路线）表示，人数用一维方向（宽度）表示，温度用一维方向（高度）表示。

同样动态信息与静态增加了时间维度，同样对于动态信息也经常用动画来表现，如西湖热点图。

交互信息与前两者有不同之处，交互信息中是两个系统之间更直接的接触（从时间上看），两个系统的外部世界重合，输入相对于对方的输出，构成一个闭环，信息在这系统之中流动。如人机交互和人人交互。下图即为人机交互的信息流图，同样当其中的计算机以人来代替时，便是人人交互，一个简单的例子是人和人的对话，对方的话（外部信息）通过听（听觉）和头脑中的记忆与知识（内部信息）组织并用语句（听觉）来传递。

在人机交互中，同样利用高维信息展示低维信息更容易被理解，但是因为交互信息设计的维度无法一直增加，所以交互信息的输入与输出维度应保持一致并受技术水平局限。当高维的输入操控低维的输出时，如用Kinect玩电视游戏，产生了大量的信息但是不能完全被传递，用户得到的信息量不足以完全支撑其下一步输入。当低维的输入操控高维的输出时，输入的信息量太低以至于输出的信息不足并难以操控，如仅用键盘操作GUI界面。

最早，输入设备是一维的键盘，输出设备也只是一维的命令行。

  • 第一次交互革命，图形交互界面出现，输出设备进入二维。
  • 第二次交互革命，鼠标取代命令行，输入设备由一维进化到二维，与二维的输出设备等维度。同时图像交互界面也伴随着输入设备的发展快速进步。但是鼠标的实际上是把一个平面上的坐标投影到了另一个平面上。或者说，以前它们分别是两个二维的平行宇宙，以一种科幻的方式连接。
  • 第三次交互革命，更应该叫第二次半革命，触屏出现，将输入坐标系和输出坐标系在二维合并。输入设备中出现触觉，输出中也更加注重交互与动态。
    后来又出现了三维的输入设备，如Wii游戏机和Kinect，但是输出设备还是二维的显示器。此时体感操控的发展还受到局限。
    再后来出现了三维的输出设备，如Oculus Rift和inForm，此时三维的交互才算真正进入可实现的状态。（JP3也算是一种音乐的三维输入与三维输出结合的方式，输入的Leap Motion，输出为变形表面。）
