数据报告撰写方法

发表于 2018-12-30
字数统计: 1.7k | 阅读时长 ≈ 5

原则

  1. 内容真实：所有数据都真实客观，所有结论都严格推论
  2. 表达准确：准确表达，没有虚词（大概、也许……）
  3. 循序渐进：一层一层逐渐深入，让人易于理解
  4. 有结果：有切实的意见和结果，能为未来提供帮助

层次

  1. 层次一 数据表达：能够提出切实的，真实可靠的数据，注意以下几点
    1. 数据需真实：有明确的定义，可追究来源，可有原始数据提供
    2. 数据需核心：提供核心数据（重要指标），是业务主要目的。而不是提供一堆数据
    3. 数据需有用：数据结果需能帮助辅助方案，有标准，如判断”太高了还是太低了“，如一个数据没有标准，则没有意义。
  2. 层次二 原因表达：能够提出数据变化的原因，注意以下几点
    1. 原因需真实：原因时真实存在的，有着严谨的推导依据，而不是”猜测“
    2. 原因需概括：原因需具体而概括，有主要原因和次要原因，而不是列出所有可能原因。
  3. 层次三 方案表达：能够为数据结果提出切实意见，帮助改进，注意一下机点
    1. 方案需目的明确：方案需明确解决上文的”原因“，而不能背离主题。
    2. 方案需具体：需提供可操作性的方案，并且能有预期结果的，而不是”可能这样更好“

内容

  1. 概要：简要表达报告主内容
    1. 事记概要：过去都做了什么重要的事情，如”10.1上线国庆节活动：充1返2“
    2. 数据概要：主要的数据有哪些，如”10月收入为123万（上月100万，23%↑）“等
    3. 原因概要：主要数据的原因时哪些，如”收入的增长主要源于赛事流量的增加（MAU……）“
    4. 意见概要：有什么样的结论和意见，为未来做指导
  2. 详细报告
    1. 数据图表：对应的数据图表的信息，需包含以下信息
      1. 图表内容：反映对应数据的表格
      2. 图表标注：重要的数据点的标注，如最高点，重要涨幅
      3. 表格详情：具体的表格内容
    2. 数据结论：根据图表所得的直接和间接的数据反馈
      1. 直接的：10月收入相比9月上涨XX%
      2. 间接的：Q3整体收入相比Q2上涨XX%
    3. 原因结论：数据表现背后的原因
      1. 被动的：有什么流量的引入，行业有什么变化等
      2. 主动的：产品运营做了什么事情，上线了什么功能
    4. 建议：根据图表获得的指导性建议，一般有
      1. 强化优势：有何可取之处可以借鉴推广
      2. 解决问题：有什么问题需要亟待解决
  3. 特殊报告，需单独说明的，有特殊的意义的内容，一般有
    1. 新上线的重要功能，如粉丝上线当月，月报中包含粉丝数据
2. 特殊的数据发现，如发现收入都来源于娱乐区，则需对娱乐区特殊分析
  4. 附录：与该数据报告相关的文档
    1. 数据报告对应表格
    2. 数据报告源数据表格
    3. 相关需求需求文档
    4. 其他与之相关的数据报告

流程

  1. 整理框架：整理所需的数据和准备呈现的形式
    1. 回顾需求：本次数据报告的目标&数据报告对应功能的目标
    2. 明确指标：将目标转化为切实的数据指标，如目标位提升收入，则指标为“每月收入值”
    3. 分类：可根据”用户“、”类型“来分类，如”用户/主播数据“、”贵族/收礼数据“，并扩展其他可能的关联指标，如和“每月收入值”相关的“每月增长率“
    4. 定义：对所有数据进行明确的定义，如”收入值为每月1号0点至次月1号零点，用户成功充值的金额“
    5. 整理：将数据整理成期望的表格和图表样式
      1. 表格：横纵轴的定义，如横轴是月份，纵轴是数据内容
      2. 图表：可能的图表表现形式，如”收入值和增长率放一张表，通过柱状图&折线图表达“
    6. 获取数据：将数据内容、数据定义期望的表格形式发给开发，由开发导出
  2. 整理表格：将源数据整理成期望的图表样式
    1. 填入数据：将开发提供的数据填入期望的表格和图表中
    2. 标注数据：在图表中进行标注，包括以下内容：
      1. 每个数据的最高点、最低点、转折点
      2. 增长率大于零和小于零的标注
      3. 重要的数据变化，如”9月增长率50%“，需在图表中明显标注
    3. 回顾数据：回顾数据表达是否有不合理之处，如有则及时修改
    4. 完善格式：标题、注释、图表的格式&大小
  3. 整理文档：将数据表格整理成文档
    1. 整理框架：按照上文所述内容的格式，梳理文档目录&框架
    2. 详细报告：
      1. 填入数据：将表格&图表内容粘贴至文档中
      2. 分析数据：罗列重点的，需要关注的数据变化
      3. 分析原因：为数据的表现寻找解释的方式
      4. 分析结论：有什么指导性的改进方案
    3. 特殊报告：需单独说明的，有特殊的意义的内容，格式与详细报告一致
    4. 概要内容：从详细报告、特殊报告中提取概要性的信息，整理成概要，内容如上文所述。
    5. 重点标注：标记重要的，需要让别人重点注意的结论
  4. 邮件发出，并附上需要的附件内容。

技巧

  1. 数据表达：多采用” XXXX（↑XXXX，XX%↑）“的方式，如”本月收入123万（↑23万，23%↑）“
  2. 语句组织：先抛数据，才写原因。如”如本月收入123万（↑23万，23%↑），原因是Ti8赛事导流，MAU达到50w（↑10w，25%↑）“
  3. 标题格式：标题直接是内容的重要结论，如该段内容都是收入，则标题为”本月收入123万（↑23w，23%↑）“
  4. 敏感词：一旦碰到和”数量“相关词语，都需写明具体数值，如”大小“、”高低“、”增加“、”大幅“等词语
  5. 数字处理：统一的数字处理格式，如
    1. 不超过4个有效数字，否则换单位
    2. 统一两个小数点
    3. 统一都有千分符
  6. 极限值标注：将每一项的极限值高亮标注，方便读者查找
  7. 立论方式：所有结论都需能立足，不写无法证实会证伪的结论
