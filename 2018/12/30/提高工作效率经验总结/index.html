提高工作效率经验总结

发表于 2018-12-30 | 分类于 经验总结
字数统计: 2.2k 字 | 阅读时长 ≈ 11 分钟

总结下来，有以下方法提高工作效率

  1. 日程安排
  2. 清单类工具的使用
  3. 文件整理习惯
  4. 效率工具的使用
  5. 良好的沟通习惯

日程安排

每日固定且合理的日程安排能够提醒人养成合理的工作习惯。
如每天上午进行当日的规划可以帮助每天内容充实。每天下午进行深度思考可以使思考效率更高效，每天晚上进行回顾总结可以避免工作疏漏。

以下为个人日程安排：

  1. 每日工作
    1. 早上
      1. 查看目标宣言
      2. 查看昨日数据
      3. 冥想&祈祷&感恩
      4. 查看日历
      5. 回顾昨日邮件&对话&纪要
      6. 查看各项目进度
      7. 安排今日工作（自己、同事）
      8. 跟进各项目进度
    2. 中午
      1. 点外卖
      2. 听每天听本书
      3. 小憩
      4. 查看行业报告&媒体
    3. 下午
      1. 跟进项目进度
      2. 思考需求
      3. 与老板沟通需求&进度
    4. 下班
      1. 查看各项目进度
      2. 今日回顾&明日计划&整理收件箱
      3. 冥想&祈祷&感恩
  2. 每周工作
    1. 周一
      1. 推进每周工作开展
    2. 周四
      1. 查看周项目进度
      2. 收集下周需求
    3. 周五
      1. 规划需求排期
      2. 与老板沟通需求安排
      3. 与成员沟通下周需求安排
    4. 周末
      1. 每周回顾&下周计划
      2. 整理文档&清理电脑

清单类工具的使用

人脑所记忆的内容极其有限，尤其在复杂的工作环境。需要“外脑”来帮助个人进行记忆，清单工具即时其中重要的一种。

以下为个人使用方式

  1. 当从外界/他人获取需完成的事情时，评估事情是否紧急，或是否能在15分钟内完成
    1. 若事情紧急，或是能在15分钟内完成则立即执行
    2. 若非上述情况，则如实记录在临时清单中
  2. 定时（每天中午/下班）回顾个人临时清单
    1. 若能当场完成，则立即执行
    2. 若不能当场完成，则记录至日程安排

文件整理的习惯

每天工作日常，会产生大量的杂乱文件。需定时对文件进行清理和整理，方便后续进行回顾和使用。

以下为个人的整理习惯，分为文件夹的使用和文件标记

  1. 文件夹使用
    1. 下载文件夹：
      1. 将所有从他人/网络上获取的文件，收入下载文件夹
      2. 每周进行文件夹清理，若有用则进入文档文件夹，若无用则删除
    2. 桌面（工作台文件夹）：
      1. 将个人工作内容（创建、下载文件）放置在桌面
      2. 任何编辑/修改内容都在桌面完成
      3. 当工作完成后，若有用则进入文档文件夹，若无用则删除
    3. 个人文档：
      1. 将所有已完成内容，收入文档文件夹
      2. 需注意文件组织格式，和文件名命名
        1. 文件组织格式：如“需求文档-增值业务-礼物改版-原型图”等
        2. 文件命名：以“【状态】日期标题/版本编辑人”，作为文件名，如“【待思考】20180808_礼物改版调研v2_xx”。需注意状态的统一性。
      3. 适时进行文档的整理和归档
  2. 重要文件标记
    1. 对经常打开的文件需进行标注，如需求List文件，项目排期文件等
    2. 将重要文件放置在方便打开的入口处

效率工具的使用

除工作必用软件外。每个人每天都会有很多频繁但复杂的操作，尽可能减少这部分操作所花时间能极大提升效率。

以下为个人整理的效率工具的使用场景：

  1. 打开App：打开常见的App如微信、浏览器等，为常见app设置快捷健操作，为非常见App设置快捷搜索
  2. 搜索文档：使用原生的Spotlight进行文件搜索
  3. 截图：使用快捷键方便截图、标记和发送
  4. 切换应用：多个应用快速切换的方式
  5. 剪切板：快速读取历史的剪切板数据
  6. 笔记：快速编辑&整理文字
  7. 文件模板生成：能快速生成常见模板如需求文档模板、需求list模板等
  8. 杀死进程/切换tab/应用内快捷键等……

良好的沟通习惯

以上均为个人能自我保持的习惯，但是工作中往往有许多和他人打交道之处。良好的沟通习惯，能帮助提升与他人共事的效率。

  1. 沟通习惯
    1. 原则：
      1. 沟通尽可能多的人：让所有人都了解到自己所需要的信息，有以下人区分
        1. 周知：仅需要了解即可，无需参与，有问题事后补充，项目边缘人员
        2. 讨论：方案确定前需听取对方意见，但无需达成一致，项目相关人员
        3. 听取：方案必须与之讨论并达成一致，项目核心人员
      2. 沟通尽可能同步：当情况有不同时，需要传达
        1. 达成一致时：如会议结束
        2. 情况变更时：如出现问题时，或较长不沟通时
      3. 沟通尽可能及时：沟通越早越好，越快越好，反馈越及时越好
        1. 尽快发起沟通：当需要沟通时立即约定
        2. 尽快提供反馈：当有结果时立即同步
      4. 沟通尽可能高效：保证双方沟通只讨论未确定内容，不浪费沟通时间
        1. 约定会议时间、地点：无需临时有变更
        2. 提早发沟通内容：沟通只需沟通要点
        3. 表述精简：内容尽可能方便阅读
        4. 背景/目的思考清楚：沟通前明确背景、目的、内容等
        5. 方式高效：视情况采用 即时通讯、面对面、邮件、会议等方式
      5. 沟通需方式/流程合理：按照正确的流程和格式进行沟通，如以下情况
        1. 项目正式进度沟通：走项目邮件流程格式
        2. 项目即时需求沟通：项目组内即时通讯
        3. 非正式讨论：私下面对面沟通
    2. 场景
      1. 多向沟通（讨论）：需要复杂交流时，如请教问题、需求讨论、需求评审
      2. 单项沟通（讲述）：当信息不同步时单方面传递
      3. 单项沟通（聆听）：听取他人的讲述
    3. 方式：
      1. 即时通讯：满足以下情况，使用即时通讯
        1. 非正式的周知性内容
        2. 短暂的讨论（3分钟内）
        3. 人数较少的沟通
      2. 会议：满足以下情况，使用会议
        1. 需要超过3分钟的沟通
        2. 较正式的沟通
        3. 参与人数较多的沟通
      3. 邮件：满足以下情况，使用邮件
        1. 较正式，低频次的沟通
        2. 作为备忘录，如进度周知等
        3. 参与人数较多的沟通
    4. 流程
      1. 沟通前：
        1. 明确会议背景、目的、内容、流程
        2. 明确会议参与人员、时间
        3. 明确会议成功条件/取消条件
        4. 参与者充分了解会议内容
      2. 沟通时：
        1. 约定进行会议记录的人
        2. 讲述会议背景、目的、内容流程、成功条件
        3. 开始会议正式内容
        4. 沟通快结束时与重复明确沟通结果
      3. 沟通后：
        1. 尽快（半小时要求）产出笔记周知群内
        2. 与大家确认是否有遗漏或错误
        3. 若正式纪要则需及时以邮件格式发出
        4. 会议内容一旦有结果/变更需及时周知
    5. 格式
      1. 问候：问候大家并简单陈述会议事项，如会议时间、会议名称
      2. 概要：以列表形式说明概要内容，需注意：
        1. 若需超过1分钟阅读，则需配以概要
        2. 概要需30秒内阅读完毕
      3. 正文：以图文形式说明正文内容（优先图片），需注意
        1. 若使用文本
          1. 单层列表若超过7项，则需再归纳
          2. 若与人相关，则需标注相关人员
          3. 若与时间相关，则需标注相关时间
          4. 若有重点，则需将重点标明
          5. 若使用附件，则需注明附件内容
        2. 若使用图片
          1. 图片需清晰明确
          2. 需标注想染他人注意的点
          3. 隐私信息需打马赛克，如数据、密码等
          4. 图标内容需有文本说明
      4. 总结性陈词：一般包含以下内容
        1. 邀请参与者阅读并回复纪要是否有误
        2. 接下来的计划安排，如下一次沟通的时间
      5. 附件：
        1. 文件命名必须包含【时间】【状态】【文件名】【发件人】【版本】
        2. 文件内容需注意格式统一规范，内容整洁。反复检查
        3. 尽量以Office或pdf格式发出
        4. 文件发出后需提醒对方已发出，请及时查收
